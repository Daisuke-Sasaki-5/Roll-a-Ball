# Roll-a-Ball  

# 使用環境  
* Untiy6  
* URP  
* 新InputSystem  
* TextMeshPro  
* 3D Physics  

# 概要  
* ボールを転がしてゴールを目指すアクションゲーム  

# 操作方法  
* WASD操作  
* SPACEでジャンプ

# ルール
* 黄色い箱のゴールオブジェクトに触れるとクリア
* ステージ外に落下すると初期位置へ戻される
* Result画面からReTry可能
